# BAH-WEBSITE
Bah-Website is a bundle that comes with all the documentation of bah.


## Documentation
The documentation can be found as marskdown inside that /pages/doc/ or can be read more graphically as web content, either by launching the server executable or by launching the app.

## Execution
- If you want to host your own version of the bah documentation, simply compile (`bah ./server.bah`) and run the `./server` file.
- If you want to browse the documentation in a window without hosting it publibly, compile (`bah ./app.bah`) and run the `./bah` file.
- If you want to rebuild some documentation files, compile (`bah ./build_doc.bah`) and run `./build_doc /opt/bah/LIB_NAME.bah`
