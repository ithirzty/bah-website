Installing the Bah compiler

Here is how to start using Bah.
To install Bah on your computer, go to the Bah Github and follow the instructions for your operating system.
We recommend that you install Git as it will make installation and updates easier.

Note for Windows

For the Bah compiler to work as intended, you will need to install Mingw GCC compiler.
In the installer, you will possibly need to install the pthreads library.

Note for MacOS

Our developping toolchain does not currently support MacOS binary, you will need to execute a command before installation. One you have ran darwin_install.sh, you will have a really out of date version of the Bah compiler. To use the latest version, run the following inside the Bah folder: 

clang -I "/opt/bah/libs/darwin/include/" -L "/opt/bah/libs/darwin/" ./installer/darwin.c -O1 -w -lgc -lpthread -lm -o /opt/bah/bah

Your first program

Make a file ending in .bah such as myFile.bah.
Then, put your Bah code inside and compile it from a terminal using: 

bah myFile.bah
This will gice you an executable (myFile or myFile.exe on windows) that can now be run.

After the file name, you can add flags. A list of all flags can be found using: 
bah -h
