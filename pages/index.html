Bah lang

Bah is a really fast multi-purpose language. It can build web servers (this website's backend was written in bah), easy and fast string manipulation, etc...
It achieves all of that without the hassle of managing your memory!
Bah can be as low level as you want, with support for C libraries (with their bindings). The compiler is fast, minimalist and all builtin libraries are cross-platform.

Install

In order to install bah-lang, follow the tutorial on github. You will need git to update bah in the future.
